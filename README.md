## 安装
> 本扩展为 TP6 开发， 请首先安装 ThinkPHP 6.0 及以上版本。

通过 Composer 安装

    composer require lunzi/top-auth

## 核心概念

### 看守器（Guard）
看守器决定如何对每个请求的用户进行身份验证。

### 提供者（Provider）
提供者决定如何从数据库中检索用户。
